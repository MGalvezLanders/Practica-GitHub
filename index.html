Lista de comandos git

      git init:inicializa la carpeta git(la hace directorio)
      git branch -m main/master:cambia a la rama que desees
      git add url:agrega los cambios realizados
      git add . :agrega todos los cambios
      git commit -m 'los cambios realizados':guarda los cambios con una descripcion
      git push -u origin main url:para mandar lo guardado a la nube
